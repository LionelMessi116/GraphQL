# Development

Thanks for contributing!

## Installing dependencies

```bash
yarn
```

## Running the demo

```sh
yarn demo
```

## Testing

```sh
yarn test            # Single pass
yarn test --watch    # Watched
yarn test --coverage # Single pass coverage report
```

## Typechecking

```sh
yarn build:ts
# or
yarn tsc
```

## Linting

```sh
yarn lint
```

## Building

```sh
yarn build
```
